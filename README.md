 Customer Shopping Behavior Data Analytics Project
** Overview**

This project performs an end-to-end analysis of customer shopping behavior to generate actionable business insights. The workflow includes data loading in Python, exploratory data analysis (EDA), data cleaning, SQL-based querying, Power BI dashboard development, and final reporting with presentation.

The goal is to understand customer purchasing patterns, segmentation, and revenue contribution to support data-driven decision-making.

==== Dataset====

The dataset contains customer transactional and shopping behavior data, including:

Customer ID

Purchase history

Product/category information

Transaction amount

Purchase frequency

Date-related fields

The data was analyzed to identify trends, customer segments, and key performance indicators (KPIs).
